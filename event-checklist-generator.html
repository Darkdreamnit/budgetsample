Welcome to Event Budget Sample!

This tool helps you create and manage checklists for any type of event. Follow this quick tutorial to learn how to use all the features.

You can always access this tutorial by clicking the "Help" button in the footer.

1. Select Your Event Type

Start by selecting one of our pre-defined event types or create your own custom event.

  • Click on any event type button to load a pre-made checklist
  • Or type your custom event name and click "Create"

The pre-made checklists include common tasks for each event type that you can customize.

2. Manage Your Checklist

Once you've selected an event type, you'll see your checklist with these features:

  • Progress Tracker: Shows your completion percentage
  • Task List: View and manage all your tasks
  • Add Task Form: Create new custom tasks

Click the checkbox next to each task to mark it as complete.

3. Adding and Editing Tasks

You can add new tasks or edit existing ones:

  • Fill out the "Add New Task" form to create tasks
  • Click "Edit" on any task to modify it
  • Click "Delete" to remove tasks

Use the sort buttons to organize tasks by priority or deadline.

4. Saving and Exporting

Save your checklist or export it for sharing:

  • Save Checklist: Stores your checklist in this browser
  • Export PDF: Download a printable PDF version
  • Email Checklist: Send to yourself or team members

Remember to save your checklist before leaving if you want to access it later!

You're Ready to Start Planning!

Now you know how to use all the features of Event Budget Sample. Here are some final tips:

  • Access your saved checklists anytime from the "Your Saved Checklists" section
  • Use the "New Checklist" button to start fresh
  • Set deadlines and priorities to stay organized

Happy planning! Your perfect event starts here.
